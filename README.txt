# Daily Push

A minimalist, time-aware reflection website built to help users pause, name their current state, and receive one strong thought at a time.

**Live site:** https://daily-push.vercel.app/

---

## About

Daily Push is a front-end website designed around one simple idea:

Instead of overwhelming the user with too much content, it gives a focused experience based on the time of day and current mood.

The site changes its tone depending on whether it is morning, afternoon, evening, or night, then asks the user what they are feeling most. After that, it shows:

- one quote
- one reflection
- one small action

The goal is not hype, but clarity, internalization, and self-reflection.

---

## Features

- Time-based mood detection
- Morning / Afternoon / Evening / Night modes
- Mood-specific prompts
- Stoic-inspired quotes and reflections
- Clean no-scroll layout
- Manual “Try another time” preview
- Fully front-end only
- Easy to edit and deploy
