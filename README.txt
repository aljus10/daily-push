DAILY PUSH — STOIC SINGLE SCREEN VERSION

FILES
- index.html
- style.css
- script.js

WHAT THIS VERSION IS
- no scroll
- time-based mood colors
- asks the likely mood first
- reveals one quote, one reflection, one action
- fully front-end only
- easy to edit in VS Code
- easy to deploy on Vercel

HOW TO RUN
1. Put these files in your daily-push folder.
2. Open the folder in VS Code.
3. Open index.html in browser or Live Server.

HOW TO DEPLOY TO VERCEL
1. Push the folder to GitHub
2. Import the repo into Vercel
3. Framework Preset: Other
4. Build Command: leave blank
5. Output Directory: leave blank
6. Install Command: leave blank

WHAT TO EDIT LATER
- modeConfig in script.js for titles and moods
- library in script.js for quotes / reflections / actions
- colors in modeConfig for the mood palette
- style.css for layout and polish
